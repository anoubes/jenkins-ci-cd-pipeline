# jenkins-ci-cd-pipeline

This project demonstrates a basic CI/CD pipeline using Jenkins Pipeline as Code.
SonarQube for scan the code 
Nexus for upload the code

## Tools Used
- Jenkins
- Git
- Cherry Tree (For Notes)

## Pipeline Stages
1. Build
2. Test
3. Deploy

## How to Use
- Create a Jenkins pipeline job
- Link it to this repository
- Jenkins will automatically execute the Jenkinsfile

## Plugins to install
- Build Timestamp
- S3 Publisher
- git
- junit
- ansicolor
- workspace cleanup
- warnings next generation
- sonarqube scanner for jenkins
- nexus artifact uploader
